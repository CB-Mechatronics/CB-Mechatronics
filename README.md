## 🛠️ Featured Project / Proyecto Destacado

### ⚡ [Motor-Predictive-Maintenance-IoT](https://github.com/CarlosJoseBA/motor_predictive)
<img width="1939" height="858" alt="Motor Predictive System 2" src="https://github.com/user-attachments/assets/ffd7375a-308a-4489-8072-8dd6d086a420" />
* **EN:** A full-stack IoT platform for predictive monitoring and fault diagnosis in three-phase industrial motors. Features high-precision 16-bit sampling via ADS1115, an isolated MPU-6050 setup, and an optimized dual-layer PCB engineered for severe EMI environments.
* **ES:** Plataforma IoT completa para el monitoreo predictivo y diagnóstico de fallas en motores industriales trifásicos. Incluye muestreo de alta precisión de 16 bits mediante ADS1115, aislamiento para el sensor MPU-6050 y una PCB de doble capa optimizada para entornos con alto ruido electromagnético.
